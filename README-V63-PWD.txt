3. Substitua os arquivos do site pelos arquivos deste ZIP.
4. Faça git add ., commit e push.

REGRAS:
- Visitantes veem somente o site público.
- Somente contas Admin/Master conseguem validar a senha.
- Abrir admin.html diretamente redireciona ao site público.
- A senha não aparece no JavaScript nem no HTML; somente salt e hash ficam no banco.
- O acesso dura apenas durante a aba/sessão atual do navegador.
